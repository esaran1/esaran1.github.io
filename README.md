# esaran1.github.io

## Editing guide
- Update profile data (name, bio, social links) in `assets/js/main.js` under `siteData`.
- Add a new blog post by copying an existing file in `blog/posts/`, then add a new card to `blog/index.html` with matching `data-*` attributes.
- Enable Pages: Settings → Pages → Deploy from branch `main` and `/ (root)`.
